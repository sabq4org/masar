# مسار — نظام إدارة العمل والمهام لغرفة الأخبار

نظام مستقل لإدارة دورة العمل التحريرية في غرف الأخبار — من الفكرة إلى النشر، بقاعدة بيانات Neon خاصة به. الخطة الكاملة (المنتج، الشاشات، الألوان، المراحل) في وثيقة «مسار» المنشورة كـ Artifact.

## التشغيل

```bash
npm install
cp .env.example .env   # واملأ DATABASE_URL و SESSION_SECRET
npm run db:push        # دفع المخطط إلى القاعدة
npm run seed           # الحالات الـ13 + الانتقالات + الأقسام + حساب المدير
npm run dev            # الخادم على 5180 والواجهة على 5181
```

الدخول الأولي: البريد المزروع في `server/seed.ts` — **غيّر كلمة المرور بعد أول دخول**.

## البنية

- `shared/schema.ts` — Drizzle، مصدر الحقيقة (13 جدولًا: users, departments, statuses, status_transitions, projects, project_sections, tasks, subtasks, task_comments, task_watchers, task_activity, notifications + جدول الجلسات التلقائي).
- `server/` — Express: `auth.ts` (جلسات + bcrypt)، `permissions.ts` (خريطة أدوار ثابتة في الكود)، `routes/` (مهام، مشاريع، بيانات مرجعية، إشعارات)، `services/tasksService.ts` (تغيير الحالة بفرض مصفوفة الانتقالات + الإشعارات + سجل النشاط)، `seed.ts`.
- `client/` — React + Vite + Tailwind، عربي RTL: دخول، مهامي (متأخرة/اليوم/قادمة/لاحقًا)، المشاريع، كانبان بالسحب والإفلات (`@dnd-kit`)، لوحة تفاصيل جانبية، مركز إشعارات.

## قرارات معمارية (MVP)

- **الحالة مصدر حقيقة واحد**: 13 حالة مزروعة بفئات سلوكية (planning/active/blocked/review/done/closed). الانتقال إلى «مؤجلة/ملغاة» مسموح من أي حالة، والخروج منهما إلى أي حالة غير مغلقة — الباقي بمصفوفة `status_transitions`.
- **مسؤول واحد لكل مهمة** + مشاركون (`task_watchers`).
- **مشروع واحد لكل مهمة** بأعمدة مباشرة (`projectId/sectionId`). الإسكان المتعدد لاحقًا عبر جدول placements بهجرة `INSERT .. SELECT` بسيطة.
- **الأدوار خريطة كود** (`server/permissions.ts`) — تنتقل لجداول عند الحاجة لأدوار مخصصة.
- **طلب التعديل في المراجعة لا يغلق المهمة** — انتقال `proofreading/awaiting_approval → editing` عادي.

## المراحل القادمة (من الخطة)

التقويم والخط الزمني، الاعتمادات، التقارير وأحمال العمل، الحقول المخصصة والقوالب والمتكررة، SSE اللحظي، محرك القواعد، الذكاء الاصطناعي (ملخص يومي، استخراج مهام من محاضر الاجتماعات)، والتكاملات (تقويم Google، بريد، ربط المواد الصحفية عبر `linkUrl` حاليًا وتكامل API لاحقًا).
